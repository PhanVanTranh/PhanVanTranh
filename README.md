## About me

I'm an electronic and telecommunication engineering student specializing in embedded systems, IoT development. Currently, I'm developing smart applications using microcontrollers and AI-based models. My work focuses on:

* **Firmware development:** C/C++ programming for ESP32, Arduino, and various sensor integrations.
* **Computer Vision:** Vehicle identification and real-time smart tracking solutions using Python and YOLO.
